Before running the Northwind tests, please make sure that:

* You have the Northwind database on your SQL Server

	- If not please see the section "Database Setup" below




Database Setup:
---------------

If you don't have the Northwind database:

(1) Download the Northwind setup from:

http://www.microsoft.com/downloads/details.aspx?FamilyID=06616212-0356-46a0-8da2-eebc53a68034&DisplayLang=en

 - or -

(1) Add a database to your SQL Server and give it the name Nortwhind.
(2) Run the script Northwind.sql against your new Northwind databasse to create the Northwind schema and fill the database with data.
